# 🎬 Movie Watchlist & Review App (Blazor .NET)

## 📌 Project Overview
This is a group project built using the .NET Blazor framework. The application will allow users to manage a movie watchlist, add reviews, and track movies they want to watch or have already watched.

## 🎯 Goals
- Build a functional Blazor web application
- Use CRUD functionality for movies and reviews
- Implement user authentication
- Deploy to a cloud platform
- Collaborate as a development team using GitHub and Trello
