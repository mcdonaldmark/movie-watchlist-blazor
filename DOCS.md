# Movie Watchlist User Guide

## Project Usercase
This Movie Watchlist application can be used by a variety of different users for them to look up movies that have come out and to leave reviews, or to read reviews from others. This will give them the option to either leave a review themselves or to add the movie to their own watchlist and then move the movie to their watched list once they've watched the movie.

## Login Documentation
When attempting to user the application the user will have to create an account. An account can be created by going to the 'Register' button when the user navigates to the website. The following info is required from the user when creating an account:
- Name
- Email
- Password

Once the user account has been successfully created the user is able to login and start using the application using the 'Login' button

## Dashboard
The dashboard of the application has a couple of different features:
- Dashboard: Button that will always bring you back to the user dashboard
- Movies: Allows you to see all of the different movies that you can choose from
- Logout: Allows the user to logout of the application once they're done
- Watchlist: Shows a number of the movies that have been put in the user watchlist. The watchlist column just below the number will actually list out the different movie cards for the user to see
- Watched Status: Will show to the user the statistics of movies that have been added to the watchlist vs those that have been watched
- Watched: Will show the movie cards of the different movies that the user has specified that they've watched

## Adding Movies
To add a movie to either the Watchlist or Watched list navigate to the movies tab on the left hand side of the screen. Once on the movies page there are different genres at the top of the screen that the user can choose from if they're looking for a spcific movie. To add the movie to the Watched list then simply select the 'Watched' button on the movie. If you're wanting to add it to your Watch List then select the 'Add to Watchlist' button. Either of these button will add them to the specified sections in the user dashboard

## Adding a Review
To add a review to a movie select the 'Reviews' button on the spcified movie car that you want. Once in the movie card you will be given the option to add a review and select a rating for the movie. Once the info is entered you will select the 'Submit' button in which case the users review will be added to the bottom of the screen.
